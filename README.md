#  AI Mock Interview Preparation Platform

An advanced AI-powered Interview Preparation Platform built using the MERN Stack with ATS Resume Analysis, AI-generated interview questions, OTP Authentication, Video Meetings, Mock Interviews, and Real-Time Interview Sessions.

---

#  Features

##  Authentication
- User Signup/Login
- JWT Authentication
- OTP Email Verification
- Forgot Password
- Secure Password Hashing

---

##  Profile Management
- Upload Profile Photo
- Change Username
- Change Email with OTP Verification
- Persistent User Sessions

---

##  AI Features
- AI Generated Interview Questions
- AI Concept Explanation
- AI Mock Interview Sessions
- AI ATS Resume Checker

---

##  Real-Time Meeting
- Video Calling using Socket.io + WebRTC
- Real-time Communication
- Live Interview Sessions

---

##  ATS Resume Checker
- Resume Upload
- ATS Score Generation
- Resume Analysis
- Skill Suggestions

---

##  Technologies Used

### Frontend
- React.js
- Tailwind CSS
- React Router DOM
- Axios
- Socket.io Client

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- Nodemailer
- Socket.io
- Multer
